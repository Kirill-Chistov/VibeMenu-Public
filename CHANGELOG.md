# Changelog

All notable, user-facing changes to VibeMenu are documented here.

## v0.2 beta

- **Claude Usage Limits Preview** — see your Claude usage limits directly from the menu bar.
- **Claude Desktop local usage detection** — reads local Claude Desktop usage data on your Mac.
- **Claude Code `statusLine` fallback** — falls back to Claude Code's status line when needed.
- **Per-limit visibility toggles** — show or hide individual limit rows.
- **Cleaner settings UI** — simplified, easier-to-scan settings.
- **Session Radar improvements** — better session state and layout.
- **Local-only privacy posture** — no telemetry, no network, no backend.
